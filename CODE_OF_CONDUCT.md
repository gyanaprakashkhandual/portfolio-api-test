# Code of Conduct

## Our Commitment

This project is maintained as an open source learning resource for the software testing and quality engineering community. Everyone who participates in this project — whether by opening issues, submitting pull requests, commenting on discussions, or contributing in any other way — is expected to uphold a respectful, constructive, and professional environment.

The maintainer is committed to making participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

---

## Standards of Acceptable Behavior

The following behaviors contribute positively to this community:

- Using clear, professional, and welcoming language in all communications
- Respecting differing opinions, approaches, and levels of experience
- Accepting constructive criticism with an open mind and a professional attitude
- Focusing discussions on the technical merits of the project and improvements to it
- Showing empathy and patience toward contributors who are learning
- Giving proper credit when referencing or building on the work of others

The following behaviors are not acceptable in this community:

- Use of sexualized, offensive, or exclusionary language or imagery
- Personal attacks, insults, or derogatory comments directed at any participant
- Public or private harassment of any kind
- Publishing private information of any individual without their explicit consent
- Trolling, sustained disruption of discussions, or deliberate provocation
- Any conduct that would be considered inappropriate in a professional environment
- Deliberate misuse of the APIs or infrastructure covered by the API Restriction Clause in the project license

---

## Scope

This Code of Conduct applies within all spaces associated with this project, including the GitHub repository, issue tracker, pull request discussions, and any external platforms where this project is formally represented.

---

## Reporting Violations

If you observe or experience behavior that violates this Code of Conduct, please report it by emailing the maintainer directly. All reports will be reviewed promptly and handled with confidentiality. The maintainer is obligated to maintain the privacy of the person reporting the incident.

Contact: Reach the maintainer through the GitHub profile at [gyanaprakashkhandual](https://github.com/gyanaprakashkhandual).

---

## Enforcement

The maintainer reserves the right to remove, edit, or reject any comments, commits, code, issues, or other contributions that violate this Code of Conduct, and to temporarily or permanently ban any contributor whose behavior is found to be harmful, disrespectful, or in breach of these standards.
